Open-Source Geshe: A Digital Debate Assistant

This website is designed to assist students of Tibetan logic and debate. Currently it has the following features:

  • A page displaying the possibilities for the comparison of phenomena.

  • An interactive page demonstrating the modes of a syllogistic reason.

  • An interactive page demonstrating correct and incorrect pervasions.

  • An interactive page demonstrating how absurd consequences flow from wrong pervasions.

  • An interactive page illustrating four possibilities.

  • An interactive page illustrating mutual exclusion.

  • An interactive page illustrating mutual inclusion.

Under construction, June 2019. William Magee, Maitripa College.
Source code available at Github.
